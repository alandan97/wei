# 🎮 围棋大师 - AI教学对弈平台

## 📖 项目简介

一个功能完整的围棋教学游戏，包含**100个教学点**，从入门到大师，帮助玩家系统学习围棋。

## ✨ 主要功能

### 1. 教学模式（100个教学点）
- **入门篇**（1-25点）：认识围棋、吃子技巧、连接切断、死活基础、基础对杀
- **进阶篇**（26-50点）：布局基础、定式入门、中盘战斗、手筋初探、官子基础
- **提高篇**（51-75点）：全局观念、定式深入、高级手筋、死活进阶、官子提高
- **大师篇**（76-100点）：AI思维、职业技巧、名局学习、专项训练、大师之路

### 2. 实时胜率计算
- 黑棋/白棋胜率实时显示
- 可视化胜率进度条
- 基于领地、提子数、棋子数综合计算

### 3. AI智能提示
- 当前教学点内容显示
- 针对性走棋建议
- 关键位置闪烁提示

### 4. 多种游戏模式
- **教学模式**：跟随100个教学点学习
- **人机对弈**：4个难度等级（入门/中级/高级/大师）
- **本地对弈**：双人对战

## 🛠️ 技术栈

- **前端**：HTML5 + CSS3 + JavaScript（原生）
- **样式**：CSS Grid + Flexbox + CSS变量
- **动画**：CSS Animation + JavaScript
- **响应式**：支持桌面端、平板端、手机端

## 📱 系统要求

- 现代浏览器（Chrome/Firefox/Safari/Edge）
- 支持JavaScript
- 屏幕分辨率建议 1200px 以上（支持响应式）
