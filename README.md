# 🎮 弈道 - 水墨围棋 v2.0

## 📖 项目简介

## 🎮 主要功能

### 1. 教学模式（100个教学点）
- **入门篇**（1-25点）：认识棋盘、吃子技巧、连接切断、死活基础
- **进阶篇**（26-50点）：布局基础、定式入门、中盘战斗、手筋初探
- **提高篇**（51-75点）：全局观念、定式深入、高级手筋、死活进阶
- **大师篇**（76-100点）：AI思维、职业技巧、名局学习、专项训练

### 2. 自动演示棋局
- **演示控制**：开始/暂停/重置按钮
- **步骤指示器**：可视化显示演示进度
- **自动落子**：每个教学点配合对应的棋局演示
- **注释说明**：每步棋都有详细的文字讲解

### 3. 实时胜率计算
- 黑棋/白棋胜率实时显示
- 胜率进度条可视化
- 基于领地、提子数、棋子数综合计算

### 4. 多种游戏模式
- **修行模式**：跟随100个教学点学习
- **人机对弈**：4个难度等级（入门/进阶/精通/大师）
- **本地对弈**：双人对战

## 🛠️ 技术栈

- **前端**：HTML5 + CSS3 + JavaScript（原生）
- **样式**：CSS Grid + Flexbox + CSS变量
- **动画**：CSS Animation + JavaScript
- **字体**：Google Fonts（站酷小薇、Noto Serif SC）
